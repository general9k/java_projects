## Library2_JPA - учебный проект, написанный на Spring MVC с использованием Spring JPA, где под капотом Hibernate.

### Задача: реализовать связь между сущностью человек и книга, установить связь один-ко-многим. Человек может принимать в свою власть всего 1 книгу. Эта книга не может быть у нескольких человек. Отображение владения книгой должно отображаться на странице у человека и у книги. Также стоит реализовать слежение за пользователем. Если он просрочил сдачу книги на 10 дней - делать на его странице подписть "просрочно" и книгу выделять красным цветом. Еще реализовать отдельную страницу для поиска книги по названию.

 - Страница для поиска книги по ее названию. В программе под это отдельный метод через Spring JPA.
